Python

Python is one of the most popular programming language in the world, but I avoided it for so long due to its speed, syntax and necessity to have dependencies everywhere. But knowledge of this skill is required in many DevOps/SRE positions. So, this is a good time to start learning it.

Flask

Usually, I create my Web app as a Discord bots. It is useful for some alerting or statistics handling, but skill in creating my own website is pretty good. It can be used for creating my own profile page or custom dashboard. Flask is an easy way of creating that kind of things, it adds more motivation to learn Python.

Docker

Docker is the de facto standard in containerization. It helps to deploy and run an application with all its dependencies, configurations and other requirements in one place. Having sound knowledge and experience about Docker is one of my professional priority. During this course, I believe I'll learn many best practices and advices about Docker and how to use it properly in real world.

DevOps

All things mentioned above will be very useful in DevOps career. In addition, working with a team project will give me extra experience and understanding of the process of developing some software in a friendly study environment. And the last, but not least - experience in coding in Vim :D
